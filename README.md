# VProUI

- VProUI 实用主义重型 Vue 3 UI 组件库

## 设计原则 / Design Principles

- 尽可能地使用原生 HTML 和较新的 JS/CSS 技术
- 部分复杂组件基于 Element Plus 实现，不重复造轮子
- 多语言无关
- 许多自动 (Auto) 组件，采用传入参数快速生成 UI
- 响应式设计，兼容各种屏幕尺寸

## 技术栈 / Tech Stack

- TypeScript
- Sass(scss)
- Vue 3 / VueUse
- ElementPlus
