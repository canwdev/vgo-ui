# VgoUI

VgoUI 组件库是一个简单、快速、轻量的 Vue 3 UI 组件库

> VgoUI is a simple, quick, lightweight Vue 3 UI library.


## 设计原则 / Design Principles:

- 尽可能地使用原生 HTML
- 一些复杂组件使用 Element Plus 实现，可以选装

## 技术栈 / Tech Stack

- TypeScript
- SCSS
- Vue 3 / VueUse
- Storybook
